defer属性

defer属性会将script脚本延迟到整个页面都解析完毕后再运行，相当于高速浏览器立即下载，但延迟执行。在现实当中，延迟脚本并不一定会按照顺序执行，也不一定会在DOMContentLoaded事件触发 前执行，因此最好只包含一个延迟脚本。

JavaScript

检测类型

typeof

调用typeof null会返回”object”，因为特殊值null被认为是一个空的对象引用。Safari5 及之前版本、Chrome7及之前版本在对正则表达式调用typeof操作符时会返回”function”，而其他浏览器在这种情况下会返回”object”。

instanceof

如果变量是给定引用类型的实例，那么instanceof操作符就会返回true。

NaN

数值转换

Number

parseInt & parseFloat

toString() & String()

String()会调用toString()并返回相应结果，但null和undefined没有toString()方法，因此直接返回”null”和”undefined”。

操作符

左移

左移不会影响操作数的符号位。

右移

带符号的右移(>>)会用符号位的值来填充所有空位，无符号右移则用0填充空位。

加减乘除

  • Infinity加-Infinity，结果是NaN
  • +0加-0，结果是+0
  • Infinity与0相乘，结果是NaN
  • Infinity与Infinity相乘，结果是Infinity
  • Infinity被Infinity除，结果是NaN
  • 如果是零被零除，结果是NaN
  • 如果是非零的有限数被零除，结果是Infinity或-Infinity，取决于有符号操作数的符号

传递参数

ECMAScript 中所有函数的参数都是按值传递的。即使在函数内部修改了参数的值，但原始的引用仍然保持未变。实际上，当在函数内部重写obj时，这个变量引用的就是一个局部对象了。而这个局部对象会在函数执行完毕后立即被销毁。
